# wordsproject
Djangoでオリジナルの英単語帳作成アプリを作りました。

## URL
(https://takkar0915.pythonanywhere.com/)


## 使い方
1.新規会員登録をする(メールアドレスは任意)  
2.ログインをする  
3.英単語の登録を行う  
4.適宜、登録した英単語の修正を行う
